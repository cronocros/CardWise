# CardWise 뱃지 및 업적 시스템 상세 명세 (BADGE_SYSTEM.md)

CardWise 사용자의 금융 활동을 장려하고 재미 요소를 더하기 위한 뱃지 및 업적 시스템의 상세 설계안입니다.

## 1. 시스템 개요
사용자의 소비 패턴, 절약 금액, 카드 이용 행태 등을 분석하여 조건 충족 시 뱃지를 부여합니다. 각 뱃지는 브론즈, 실버, 골드, 플래티넘의 4단계 등급을 가집니다.

## 2. 뱃지 카테고리 및 획득 조건

### 2.1. 소비 마스터 (Spending Master)
- **주제**: 현명한 소비 및 특정 카테고리 이용
- **등급별 조건**:
    - **커피 중독 (Bronze)**: 카페 카테고리 5회 이용
    - **커피 매니아 (Silver)**: 카페 카테고리 15회 이용
    - **커피 마스터 (Gold)**: 카페 카테고리 30회 이용
    - **카페 정복자 (Platinum)**: 카페 카테고리 50회 이용 + 특정 카드(커피 특화) 이용

### 2.2. 절약의 왕 (Saving King)
- **주제**: 카드 혜택(할인/적립) 극대화
- **등급별 조건**:
    - **절약의 시작 (Bronze)**: 누적 할인 금액 10,000원 돌파
    - **현명한 생활 (Silver)**: 누적 할인 금액 50,000원 돌파
    - **연회비 회수 완료 (Gold)**: 누적 할인 금액 150,000원 돌파
    - **재테크 신 (Platinum)**: 누적 할인 및 포인트 적립 총액 500,000원 돌파

### 2.3. 카드 파인더 (Benefit Hunter)
- **주제**: 다양한 카드 혜택 및 바우처 이용
- **등급별 조건**:
    - **초보 사냥꾼 (Bronze)**: 첫 바우처 이용
    - **혜택 수집가 (Silver)**: 3개 이상의 카드 혜택 고르게 이용
    - **바우처 매니아 (Gold)**: 누적 바우처 이용 10회 돌파
    - **카드 와이즈 (Platinum)**: 모든 등록 카드의 1단계 실적 매달 달성 (3개월 연속)

## 3. 시각적 등급 가이드
- **브론즈**: 은은한 구리빛 메탈릭 질감, 3D 매트 스타일
- **실버**: 세련된 은색 광택, 글래스모피즘(Glassmorphism) 배경 조합
- **골드**: 화려한 금빛 그라데이션, 후광(Glow) 효과 적용
- **플래티넘**: 다이아몬드 빛 반사 효과, 프리미엄 다크 테마 전용 뱃지

## 4. UI/UX 반영 계획
1.  **배지 갤러리**: 마이페이지에서 전체 뱃지 현황(획득/미획득)을 3D 그리드로 노출
2.  **획득 알림**: 조건 충족 시 `AchievementModal`을 통해 축하 효과(폭죽, 3D 회전 애니메이션)와 함께 노출
3.  **홈 위젯**: 가장 최근에 획득한 프리미엄 뱃지를 홈 화면 프로필 옆에 작게 노출

## 5. 데이터 구조 (Proposed)
```typescript
interface Badge {
  id: string;
  name: string;
  category: 'spending' | 'saving' | 'benefit';
  tier: 'bronze' | 'silver' | 'gold' | 'platinum';
  achievedDate?: string;
  description: string;
  icon: string; // Emoji or SVG path
  progress?: number; // 0 to 100
}
```
