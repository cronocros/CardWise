# Design Prompts Guide

This document maintains the unified prompts for generating consistent assets and components for CardWise.

## 1. Mascot: Kkulsori (Honey Badger)
**Base Prompt:**
> "A cute, premium 2D/3D flat-style honey badger character named Kkulsori. High-end fintech mascot style. Two-tone fur: charcoal black lower body/limbs and snow-white upper body/head. Large expressive eyes, friendly smile. Clean lines, minimalist shading. Soft pastel highlights. Background is flat white."

**Variations:**
- **Waving**: "Kkulsori mascot waving one hand enthusiastically, greeting the user. High-end app illustration style."
- **Celebrating**: "Kkulsori mascot with arms raised in victory, surrounded by subtle confetti. Celebrating a financial goal achievement."
- **Thinking**: "Kkulsori mascot with a hand on its chin, looking thoughtful or curious. Neutral but friendly expression."
- **Warning**: "Kkulsori mascot looking slightly worried, holding an empty wallet or a 'budget over' sign. Soft crying expression."

## 2. App Icons
**Icon Pack Prompt:**
> "Uniform set of fintech app icons for CardWise. Rounded square (Superellipse) containers. Soft cream background. Main icon elements use the charcoal black and rose-400 (#fb7185) color palette. Theme: 'Smart Financial Management'. High-quality vector style, 3D claymorphism influences but kept flat for mobile performance."

## 3. UI Backgrounds
**Rose Glass (Dark) Prompt:**
> "Premium dark fintech background. Deep navy/plum base (#080010). Vibrant glassmorphism blobs in rose-pink and violet (#f43f5e, #8b5cf6). 40% opacity, 120px Gaussian blur. Subtle grain texture. Modern, sleek, and high-end."

**Rose Blossom (Light) Prompt:**
> "Soft, airy fintech background. Off-white/rose-50 base (#fff1f2). Subtle gradient of pale pink highlights. Clean, friendly, and inviting."
