# CardWise 카드 브랜드 및 등급 체계 명세 (CARD_SPEC_V4.md)

CardWise v4.0에서는 글로벌 스탠다드에 맞춘 해외 제휴 브랜드 및 카드 등급 체계를 도입하여 데이터의 전문성과 시각적 완성도를 높입니다.

## 1. 해외 제휴 브랜드 (International Brands)
모든 카드는 다음 중 하나의 해외 제휴 브랜드를 가질 수 있으며, 카드 전면에 해당 로고가 고해상도로 표시됩니다.

- **Visa**: 글로벌 점유율 1위, 범용성 최상
- **Mastercard**: 전 세계 어디서나 안정적인 결제
- **American Express (Amex)**: 프리미엄 서비스 및 특화 혜택
- **UnionPay (CUP)**: 아시아권 특화 혜택
- **JCB**: 일본 및 아시아 태평양 지역 혜택
- **Diners Club**: 프리미엄 라운지 및 여행 혜택

## 2. 카드 등급 체계 (Card Tiers)
카드별 혜택의 수준을 결정하는 등급 체계를 다음과 같이 정립합니다.

| 등급 (Tier) | 특징 | 시각적 표현 |
| :--- | :--- | :--- |
| **Classic/Standard** | 기본 혜택 중심 | 깔끔한 단색 또는 심플 그라데이션 |
| **Gold** | 우대 혜택 제공 | 골드 엠블럼 및 금릿 반사 효과 |
| **Platinum** | 프리미엄 서비스 | 플래티넘 로고, 글래스모피즘 효과 |
| **Signature/World** | 최상위 고객 대상 | 고급스러운 텍스처, 은은한 발광 효과 |
| **Infinite/World Elite** | VVIP 초프리미엄 | 다이아몬드 포인트, 다크 프리미엄 테마 |

## 3. 카드 상호작용 (Flip Interaction)
- **앞면 (Front)**: 브랜드 로고, 카드 번호(마스킹), 만료일, 등급 라벨 노출.
- **뒷면 (Back)**: 카드 이미지 클릭 시 CSS 3D Flip 애니메이션과 함께 전환. 해당 카드의 **TOP 3 핵심 혜택**과 **카드 관리 메뉴** 노출.

## 4. 데이터 연동 (Sample Data Update)
`sampleData.ts`의 `Card` 인터페이스에 `brand` 및 `tier` 필드를 추가하고, 모든 샘플 카드에 적합한 값을 할당합니다.
